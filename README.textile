h1. Welcome to Prometheus 2.0

h2. Setup and run

h3. Install:
* Ruby EE 1.8.7 2011.03
* ruby-rpm
* bundler
* PostgreSQL 9.0 (with other databases it doesn't tested)
* nginx + passenger
* gettext
* sphinx search engine

h3. And then:
* run "bundle install" in project folder

h3. Prepare:
* rake gettext:pack

h3. Init database:
