h1. Welcome to Prometheus 2.0

!https://gemnasium.com/biow0lf/prometheus2.0.png(Dependency Status)!:https://gemnasium.com/biow0lf/prometheus2.0

h2. Setup and run

h3. Install:

* Ruby 1.9.3-p0
* RubyGems 1.8.12 (you will need install slimgems for gettext:find and gettext:pack)
* bundler 1.1.rc.5
* PostgreSQL 9.1 (with other databases it doesn't tested)
* nginx
* gettext
* curl
* sphinx search engine
* node.js
* /usr/bin/md5sum from coreutils
* git
* memcached
* redis

h3. And then:

* run "bundle install" in project folder

h3. Prepare:

* bundle exec rake gettext:pack

h3. Init database:

* bundle exec rake db:create
* bundle exec rake db:schema:load

* bundle exec rake db:seed

* bundle exec rake sisyphus:srpms
* bundle exec rake sisyphus:binary
* bundle exec rake sisyphus:acls
* bundle exec rake sisyphus:leaders
* bundle exec rake sisyphus:teams

* bundle exec rake sisyphus:bugs
* bundle exec rake sisyphus:repocops
* bundle exec rake gear:import
* bundle exec rake perlwatch:update
* bundle exec rake ftbfs:update

* bundle exec rake sisyphusarm:srpms
* bundle exec rake sisyphusarm:binary

* bundle exec rake platform6:srpms
* bundle exec rake platform6:binary

* bundle exec rake t6:srpms
* bundle exec rake t6:binary

* bundle exec rake platform5:srpms
* bundle exec rake platform5:binary

* bundle exec rake 51:srpms
* bundle exec rake 51:binary
* bundle exec rake 51:acls
* bundle exec rake 51:leaders
* bundle exec rake 51:teams

* bundle exec rake 50:srpms
* bundle exec rake 50:binary
* bundle exec rake 50:acls
* bundle exec rake 50:leaders
* bundle exec rake 50:teams

* bundle exec rake 41:srpms
* bundle exec rake 41:binary
* bundle exec rake 41:acls
* bundle exec rake 41:leaders
* bundle exec rake 41:teams

* bundle exec rake 40:srpms
* bundle exec rake 40:binary
* bundle exec rake 40:acls
* bundle exec rake 40:leaders
* bundle exec rake 40:teams

* bundle exec rake redis:cache

* bundle exec rake ts:index

h2. Run

* bundle exec rake ts:start
* bundle exec rake assets:precompile
* redis-server config/redis.conf
* service postgresql start
* memcached -d -m 128 (as a user)

h2. Deploy

* cap deploy
