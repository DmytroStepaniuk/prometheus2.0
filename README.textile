h1. Welcome to Prometheus 2.0

h2. Setup and run

h3. Install:

* Ruby 1.9.2p290
* RubyGems 1.8.8
* ruby-rpm
* bundler 1.1.pre.7
* PostgreSQL 9.0 (with other databases it doesn't tested)
* nginx + passenger
* gettext
* sphinx search engine
* /usr/bin/md5sum from coreutils

h3. And then:

* run "bundle install" in project folder

h3. Prepare:

* rake gettext:pack

h3. Init database:

* rake db:seed
